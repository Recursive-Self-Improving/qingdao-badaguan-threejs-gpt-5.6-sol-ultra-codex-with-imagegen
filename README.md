# 青島八大關 · Three.js 互動虛擬景觀

以程序化幾何、Canvas 材質和 Three.js 即時光影重構八大關南部的空間印象：成熟法桐林蔭、低密度花園別墅、花石樓、公主樓，以及道路盡頭的太平灣與第二海水浴場。

## 啟動

```bash
npm install
npm run dev
```

正式建置與檢查：

```bash
npm run check
npm run build
npx playwright install chromium  # 首次執行瀏覽器測試時
npm run smoke
```

## 操作

- `W A S D`／方向鍵：移動
- 滑鼠：環視
- `Shift`：快走
- `E`：閱讀附近地標
- `M`：地圖
- `H`：操作說明
- `Esc`：釋放滑鼠

觸控裝置提供左側虛擬搖桿與右側拖曳環視。聲景預設關閉，需由使用者主動開啟。

## 視覺取材

場景不是精確測繪或建築復原，而是依八大關真實的城市尺度與辨識語彙所做的互動濃縮：建築保持低層、退縮於庭院內，以紅瓦、花崗岩基座、粉牆和多國近代別墅輪廓組成；植被以法桐、松柏和少量初秋色葉為主；海面採低飽和藍灰，保留青島沿岸的濕潤薄霧。
